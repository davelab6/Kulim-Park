## Kulim Park

Kulim Park is a humanist sans serif typeface, with high x-height, open counter ‘a’, minimal degrees of contrast in stem width, inviting bowls and a design language aimed at encapsulating openness.

### Available weights
Extra Light  
Light  
Light Italic  
Regular  
Bold  
Extra Bold
